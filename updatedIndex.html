Skills

Html

HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications. It defines the structure and layout of content on a webpage using a system of tags and attributes.

CSS 3

CSS 3 is a language used for describing the presentation of a document written in HTML or XML including layout, colors, and fonts. It enhances the visual appeal and design consistency of web pages across different devices and screen sizes.

Java Script

JavaScript is a high-level, interpreted programming language primarily used to add interactivity and dynamic behavior to web pages. It enables developers to create interactive elements, handle user input, and manipulate webpage content dynamically.

PHP

PHP (Hypertext Preprocessor) is a widely-used open-source scripting language that is particularly suited for web development and can be embedded into HTML. It's commonly used for server-side scripting to generate dynamic web page content and manage databases.

SASS

Sass (Syntactically Awesome Style Sheets) is a preprocessor scripting language that helps streamline and enhance the functionality of CSS. It offers features like variables, nesting, and mixins, making CSS code more efficient and maintainable for web development projects.

GitHub

GitHub is a web-based platform for version control and collaboration using Git. It provides tools for hosting, reviewing, and managing code repositories, facilitating teamwork among developers in software projects.

Node Js

Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to run JavaScript code outside of a web browser. It's commonly used for building scalable network applications and server-side programming.

React Js

React.js is a JavaScript library used for building user interfaces, particularly for single-page applications. It allows developers to create reusable UI components that efficiently update when data changes, providing a smoother and more interactive web experience.

Vercel

Vercel is a cloud platform for deploying and hosting static websites and serverless functions with ease. It offers features like automatic deployments, instant scaling, and integration with popular frontend frameworks like Next.js and React.

Vue.Js

Vue.js is a progressive JavaScript framework for building interactive user interfaces and single-page applications. It emphasizes simplicity and flexibility, allowing developers to easily integrate it into existing projects and scale as needed.

Tailwind CSS

Tailwind CSS is a utility-first CSS framework that provides pre-defined classes for styling web interfaces. It enables rapid development by abstracting common design patterns and allowing for easy customization through utility classes.

Laravel

Laravel is a popular open-source PHP web framework known for its elegant syntax and developer-friendly features. It simplifies the process of building web applications by providing tools for routing, authentication, database management, and more.

Figma

Figma is a collaborative interface design tool used for creating, prototyping, and collaborating on digital designs in real-time. It enables designers and teams to work together seamlessly, streamlining the design process from concept to implementation.

Firebase

Firebase is a comprehensive platform offered by Google for developing web and mobile applications, providing features such as real-time database, authentication, hosting, cloud functions, and more, all managed through a unified console.

MySQL

MySQL is an open-source relational database management system (RDBMS) known for its reliability, scalability, and ease of use. It allows users to store, organize, and retrieve data efficiently, making it widely used in various web applications and software projects.

Boot Strap

Bootstrap is a popular front-end framework for building responsive and mobile-first websites and web applications. It provides pre-designed CSS and JavaScript components, facilitating rapid development and ensuring consistency in design across different devices.

JQuery

jQuery is a fast, small, and feature-rich JavaScript library designed to simplify HTML document traversal and manipulation, event handling, and animation. It is widely used for creating interactive web experiences with less code and greater efficiency.

Shopify

Shopify is an e-commerce platform that allows businesses to create online stores and sell products seamlessly. It provides customizable templates, secure payment processing, inventory management, and various marketing tools to help merchants grow their online presence.

Word Press

WordPress is a popular open-source content management system (CMS) used for building websites and blogs. It offers a user-friendly interface, extensive plugin ecosystem, and customizable themes, making it accessible for beginners and powerful for developers.

Web Flow

Webflow is a powerful web design platform that allows users to visually design and build responsive websites without writing code. It offers features for design, prototyping, and hosting, empowering designers to create sophisticated websites with ease.
